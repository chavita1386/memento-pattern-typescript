Memento Pattern

Problem

Imagine you have a some component that you use a lot, and from time to time you change the state of this component, but what happen when you need to back the state from this component, this is where the Memento pattern comes in.

Example

In this example we're going to see how to implement the Memento Pattern. We have an input text and we'll be able to change its styles and its content and we have also two additional buttons to save and restore the state.
